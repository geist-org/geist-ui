<br/>

## Quick Start

1. run `yarn add @zeit-ui/react` or `npm i @zeit-ui/react` install it.

2. import into project:

```jsx
import { ZeitProvider, CssBaseline } from '@zeit-ui/react'

const Application = () => (
  <ZeitProvider>
    <CssBaseline /> // ---> normalize styles
    <AppComponent /> // ---> Your App Component
  </ZeitProvider>
)
```
