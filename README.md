## Project Status

It is time to say goodbye. Geist UI was created around CSS-in-JS, and many
of its components depend on styled-jsx for the kind of dynamic styling that
made this project flexible, expressive, and enjoyable to maintain for a long
time.

Today, it is also reasonable to recognize that CSS-in-JS still leaves
room for improvement in performance, scalability, and AI-friendly semantics.
Tailwind CSS has become a practical and widely adopted choice for modern
interface work, and it is the direction I have personally used most in recent
years.

Geist UI will be archived and will no longer receive active updates.
Thank you to every developer who has followed, used, discussed, and contributed
to this project.
