## 介绍

### 什么是 ZEIT UI - React

`@zeit-ui/react` 是对 <Link rel="nofollow" target="_blank" pure href="https://github.com/zeit-ui/zeit-style">ZEIT 样式库</Link> 的一个 React 版本实现，
灵感来自于 <Link rel="nofollow" target="_blank" pure href="https://zeit.co/design">ZEIT 的设计稿</Link>。

ZEIT 的设计非常简洁美观，且有浓厚的 Geek 风格，这是 ZEIT 如此受欢迎的重要原因，现在你可以在 React 中随意使用所有设计元素。

ZEIT UI 是非官方的组织，且所有项目都是开源的。
