<Attributes edit="/pages/zh-cn/components/link.mdx">
<Attributes.Title>Link.Props</Attributes.Title>

| 属性 | 描述 | 类型 | 推荐值 | 默认
| ---------- | ---------- | ---- |  -------------- | ------ |
| **href** | 链接地址 | `string` | - | - |
| **color** | 是否高亮显示 | `boolean` | - | `false` |
| **icon** |  在文本后显示图标 | `boolean` | - | `false` |
| **underline** | 显示下划线 | `boolean` | - | `false` |
| **block** | 是否以块的方式显示链接 | `boolean` | - | `false` |
| ... | 原生属性 | `AnchorHTMLAttributes` | `'rel', 'target', ...` | - |

<Attributes.Title>NextLink.Props</Attributes.Title>

| 属性 | 描述 | 类型 | 推荐值 | 默认
| ---------- | ---------- | ---- |  -------------- | ------ |
| **href** | NextJS 路由属性 | `string` / `UrlObject` | - | - |
| **as** | NextJS 路由属性 | `string` / `UrlObject` | - | - |
| **replace** | NextJS 路由属性 | `boolean` | - | `false` |
| **scroll** | NextJS 路由属性 | `boolean` | - | `true` |
| **shallow** | NextJS 路由属性 | `boolean` | - | `false` |
| **passHref** | NextJS 路由属性 | `boolean` | - | `true` |
| **prefetch** | NextJS 路由属性 | `boolean` | - | `true` |
| **...** | Link 组件属性 | `LinkProps` | [LinkProps](#link.props) | - |
