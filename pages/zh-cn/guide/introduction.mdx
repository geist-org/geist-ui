## 介绍

### 关于

**Geist UI** 是以 <Link rel="nofollow" target="_blank" href="https://github.com/geist-org/style">Geist 样式库</Link> 为基础实现的 React 版本，
灵感来自于 <Link rel="nofollow" target="_blank" href="https://vercel.co/design">Vercel 的设计稿</Link>。

Vercel 的设计非常简洁美观，且有浓厚的 Geek 风格，这是 Vercel 如此受欢迎的重要原因。现在你可以在
React 中随意使用所有设计元素。

Geist UI 一个用于构建现代网站和应用程序的开源设计系统，且所有项目都是开源的。
