<Attributes edit="/pages/en-us/components/tag.mdx">
<Attributes.Title>Tag.Props</Attributes.Title>

| Attribute    | Description                                                  | Type                 | Accepted values                | Default   |
| ------------ | ------------------------------------------------------------ | -------------------- | ------------------------------ | --------- |
| **variant**  | tag variant                                                  | `TagVariants`        | [TagVariants](#buttonVariants) | `default` |
| **color**    | tag color                                                    | `TagColors`          | [TagColors](#buttontypes)      | `default` |
| **size**     | tag size                                                     | `TagSizes`           | [TagSizes](#tagsizes)          | `medium`  |
| **toggable** | make tag toggable                                            | `boolean`            |                                | `false`   |
| **effect**   | display animation                                            | `boolean`            | -                              | `false`   |
| **disabled** | disable tag                                                  | `boolean`            | -                              | `false`   |
| **dashed**   | tag with dashed border, only works if `variant` is `default` | `boolean`            | -                              | `false`   |
| **onClick**  | click handler                                                | `MouseEventHandler`  | -                              | -         |
| ...          | native props                                                 | `SpanHTMLAttributes` | `'id', 'className', ...`       | -         |

<Attributes.Title>TagVariants</Attributes.Title>

```ts
type TagVariants = 'default' | 'solid' | 'text'
```

<Attributes.Title>TagColors</Attributes.Title>

```ts
type TagColors = 'default' | 'primary' | 'secondary' | 'success' | 'warning' | 'error'
```

<Attributes.Title>TagSizes</Attributes.Title>

```ts
type TagSizes = 'small' | 'medium' | 'large'
```

</Attributes>
